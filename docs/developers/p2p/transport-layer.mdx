# Transport Layer

We use [QUIC](https://en.wikipedia.org/wiki/QUIC) as our transport layer for peer to peer communication. QUIC is the perfect protocol for our usecases as it's fast, has built in stream multiplexing and has TLS built in so we can encryption out of the box.
