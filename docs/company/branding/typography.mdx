## Font Family

We use a modern and accessible font stack designed to ensure readability and performance across all platforms and devices:

```css
font-family: ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
```
